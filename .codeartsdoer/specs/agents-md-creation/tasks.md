### 任务描述

### 输出

- 文件包含完整的章节标题结构

### 验收标准
- [ ] AGENTS.md 文件存在于项目根目录
- [ ] 文件编码为 UTF-8 without BOM
- [ ] 文件包含 7 个主要章节标题
- [ ] Markdown 格式正确，可被解析器正确渲染

### 代码生成提示
```

1. 概述
2. 文件编码规范

5. 文件组织规范
6. 版本控制规范

### 代码生成提示
```
